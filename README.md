# Forensic Scanner - Античит Сканер

Десктопная программа для локального анализа системы пользователя на наличие признаков использования читов и запрещённых модификаций.

## Возможности

- **Анализ хешей файлов**: Сканирование системных директорий и сравнение хешей с базой известных читов
- **Проверка истории браузера**: Поиск посещений подозрительных сайтов и ключевых слов
- **Анализ Discord активности**: Обнаружение участия в каналах, связанных с читами
- **Графический интерфейс**: Удобный UI с индикатором прогресса и цветовой оценкой риска
- **Детальное логирование**: Подробные логи по каждой категории проверок

## Требования

- Windows x64
- CMake 3.16+
- Qt6 (Core, Widgets)
- Компилятор C++17 (MSVC, MinGW, Clang)

## Быстрая сборка

### Windows (Visual Studio)

```batch
build.bat
```

### Ручная сборка

1. Установите зависимости (см. BUILD_INSTRUCTIONS.md)
2. Создайте папку build:
   ```bash
   mkdir build
   cd build
   ```
3. Настройте CMake:
   ```bash
   cmake .. -G "Visual Studio 17 2022" -A x64 -DQt6_DIR="путь/к/Qt6"
   ```
4. Соберите проект:
   ```bash
   cmake --build . --config Release
   ```

## Использование

1. Запустите `ForensicScanner.exe`
2. Нажмите "Начать сканирование"
3. Дождитесь завершения сканирования
4. Просмотрите результаты в логах

## Конфигурация

Отредактируйте файлы в папке `config/`:
- `config.json` - основные настройки (ключевые слова, домены, веса риска)
- `hash_database.txt` - база хешей известных читов
